# How to Auto-Fit Column Widths in CSV Files

## In Microsoft Excel:
1. Open the CSV file
2. Select all columns (Click column A header, then press `Ctrl + Shift + Right Arrow`)
3. **Method 1**: Double-click the border between any two column headers
4. **Method 2**: Go to Home → Format → AutoFit Column Width

## In Google Sheets:
1. Open the CSV file
2. Select all columns (Click the box in top-left corner above row 1)
3. Right-click any column header
4. Click "Resize columns A-Z"
5. Select "Fit to data"

## In LibreOffice Calc:
1. Open the CSV file
2. Select all columns (`Ctrl + A`)
3. Go to Format → Columns → Optimal Width
4. Click OK

## Quick Tip:
The CSV file data is already clean and well-formatted. The column widths appear narrow initially because CSV is a plain text format. Once you auto-fit in your spreadsheet application, all data will be perfectly visible!
